<!-- GETTING STARTED -->
## Getting Started

#### Prerequisite

* Python 3.7

#### Instruction

1. Clone the repo or download the zip
```sh
git clone https://github.com/henry32144/draw-lots.git
```
2. Install the libraries
```sh
cd draw-lots

pip install flask
```
3. Run Flask server
```sh
python app.py
```
4. Open your browser and type `127.0.0.1:5000` in url
